# Beispieldateien

Beispieldateien werden aus datenschutzrechtlichen Gründen nicht im Repository gespeichert.

## Erwartetes Format: Zeitplan

Excel-Datei mit Jahresblättern (`1. Jahr`, `2. Jahr`, `3. Jahr`, ...).  
Jedes Blatt enthält ein tagesgenaues Kalenderraster:

| Zellwert | Bedeutung |
|----------|-----------|
| leer / Zahl | Unterrichtstag |
| `Fe` | Ferien |
| `Pr` | Praktikum |
| `Prüf` | Prüfung |
| Wochenende | kein Eintrag |

Zusammenfassungszeilen (~Zeile 37–44) enthalten Monatsstatistiken (Theorie-Tage, UE, Praktikumstage, Ferientage).

## Erwartetes Format: Modulplan

Excel oder Word mit mindestens diesen Spalten:

| Spalte | Beispiel |
|--------|---------|
| Modul | `1`, `IMMO 1` |
| Modulbezeichnung | `Volkswirtschaftliche Grundlagen` |
| UE | `90` |
